What is "The PKFR Spot"?

We aim to be the spot for all PKFR things. We are integrating lots of different features and capabilities into this website to make it a great tool for freerunners and traceurs. We want to provide a hub to let freerunners share media, scout locations, plan sessions and grow as a community.

Is there a catch?

Well, we are running ads. But we are planning to collaborate with parkour brands and gyms to only show you relevant PKFR-related ads.

Also, maybe we will be accepting donations in the future for future projects, like an app, for example.

Our source code is public

All our source code is public and available on GitHub.
